# Statystyki

Widok statystyk pokazuje, jak reputacja firmy zmienia się w czasie.

- Liczba nowych opinii w wybranym okresie.
- Średnia ocena oraz jej trend.
- Dynamika pozyskiwania opinii.
- Porównanie okresów i, docelowo, lokalizacji.
- Czytelne wskaźniki bez przeciążania panelu danymi.
