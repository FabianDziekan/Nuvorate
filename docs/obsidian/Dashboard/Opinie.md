# Opinie

Widok opinii pozwala przeglądać, filtrować i obsługiwać informacje zwrotne od klientów.

- Lista opinii z oceną, datą, źródłem i treścią.
- Filtrowanie według oceny, okresu oraz statusu.
- Szybkie przejście do opinii wymagających reakcji.
- Oznaczanie opinii jako przeczytanych lub obsłużonych.
- Powiązanie pojedynczej opinii z wynikami analizy.
