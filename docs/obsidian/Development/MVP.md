# MVP

MVP NuvoRate ma potwierdzić, że firmy chcą systematycznie zbierać i monitorować opinie w modelu abonamentowym.

- Rejestracja, logowanie i podstawowa konfiguracja firmy.
- Widok opinii oraz podstawowe statystyki.
- Obsługa planu Starter i płatności cyklicznych.
- Prosty proces prowadzący klienta do wystawienia opinii.
- Konfiguracja plakietki NFC jako dodatku.
- Zakres MVP nie powinien obejmować wszystkich docelowych analiz.
