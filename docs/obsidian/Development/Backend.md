# Backend

Backend NuvoRate obsługuje dane firm, opinie, abonamenty, powiadomienia i procesy analityczne.

- Bezpieczne zarządzanie kontami, firmami i uprawnieniami.
- Przechowywanie opinii wraz ze źródłem, datą i statusem.
- Integracja płatności cyklicznych dla planów Starter i Business.
- Obsługa konfiguracji NFC oraz przekierowań.
- Mechanizmy powiadomień i inteligentnej analizy.
- Należy uwzględnić prywatność danych, audyt i niezawodność integracji.
