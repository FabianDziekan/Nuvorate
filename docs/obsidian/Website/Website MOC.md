# Website MOC

Mapa wiedzy dla strony publicznej i komunikacji landing page.

## Struktura strony

- [[Website Blueprint]]: pełna struktura strony głównej.
- [[Hero]]: pierwszy ekran i główny przekaz.
- [[Jak działa]]: przepływ użytkownika i opinii.
- [[Funkcje]]: gotowe, częściowe i planowane funkcje.
- [[Cennik]]: ceny, limity i Stripe.
- [[FAQ]]: odpowiedzi zgodne z aktualnym kodem.
- [[CTA]]: wezwania do działania.

## Zależności

- [[Brand MOC]]: styl, kolory i język.
- [[Product MOC]]: plany i funkcje.
- [[Dashboard MOC]]: dashboard jako główny element prezentowany na stronie.
- [[Sales MOC]]: argumentacja sprzedażowa.
