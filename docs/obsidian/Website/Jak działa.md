# Jak działa

Sekcja „Jak działa” opisuje prostą drogę od uruchomienia NuvoRate do wzrostu reputacji firmy.

- Firma zakłada konto i konfiguruje profil.
- Klienci otrzymują prostą możliwość wystawienia opinii, także przez NFC.
- NuvoRate zbiera dane i pokazuje je w jednym panelu.
- Powiadomienia pomagają szybko reagować.
- Analiza zamienia opinie w praktyczne wskazówki.
