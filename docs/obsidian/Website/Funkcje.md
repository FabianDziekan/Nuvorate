# Funkcje

Sekcja funkcji prezentuje możliwości platformy przez pryzmat korzyści dla firmy.

- Zbieranie i porządkowanie opinii.
- Monitoring reputacji w jednym miejscu.
- Powiadomienia o nowych ocenach.
- Statystyki oraz inteligentna analiza treści.
- Obsługa dodatków NFC ułatwiających klientom działanie.
