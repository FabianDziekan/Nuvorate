# NuvoRate Hub

Centralny punkt bazy wiedzy NuvoRate, platformy do zarządzania opiniami i reputacją online.

- Główna zależność wartości: więcej opinii = większe zaufanie = więcej klientów.
- Podstawowym produktem jest abonament Starter lub Business.
- Plakietki NFC są dodatkiem ułatwiającym klientom wystawianie opinii.
- Dokumentacja obejmuje strategię, markę, produkt, sprzedaż i rozwój platformy.
